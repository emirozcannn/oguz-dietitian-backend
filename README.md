# Dietitian Backend

Bu klasör, Oğuz Yolyapan diyetisyen platformunun Node.js/Express tabanlı backend API kodlarını içerir.

## Kurulum

```bash
npm install
```

## Çalıştırma

```bash
npm start
```

## Ortam Değişkenleri

Gerekli .env dosyası için örnek:

```
MONGODB_URI=your_mongodb_connection_string
PORT=3000
```

## Ana Dosyalar
- server.js
- routes/
- src/
- package.json
